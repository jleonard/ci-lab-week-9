Breakpoints

In the css/breakpoints.css file you'll find the .css you need for this page. The css includes 4 breakpoints for small devices and medium, large and x-large viewports.

Assignment

Style this page so that...

  1. Users with small devices always see red background color on the page and users on desktops never see a red background.
  2. Desktop users see a black background for any viewport width from medium up.
  3. Only users with large and x-large viewports see the cat image.

Note if you open the chrome web inspector you'll see the pixel dimensions of your browser window as you resize
