Getting Setup

We're going to use a server this week so we can test the lab examples on our phones. This will only work if your phone is on the same wifi network as your laptop.

  1. Open a Terminal window.
  2. Type cd and then drag and drop this project folder into the terminal. Hit Enter.
  3. Paste this into the Terminal and hit Enter to start the server. Try visiting http://localhost:8080 in your browser to confirm that it is working. 
    
python -m SimpleHTTPServer 8080
  4. Open the Settings on your Mac and navigate to Network. You'll see your computer's IP address under the WIFI settings. On your phone, enter that address and hit enter. If you're successful you'll see this page.
